Chào Mừng Đến Với Lab3

Trang tĩnh cực “đẹp đẹp” được xây dựng bằng Flask & Bootstrap

Tìm Hiểu Ngay

Jenkins Lab8

Lab3 là một ví dụ về cách bạn có thể tạo nên một landing page tĩnh nhưng vẫn cực “bắt mắt”, responsive bằng Flask cho phần backend và Bootstrap cho phần frontend. Mục đích là giúp bạn nắm vững cách tổ chức file, deploy lên GitHub & Render/Heroku một cách nhanh chóng.

  • Sử dụng Flask để phục vụ file tĩnh
  • Bootstrap 5 cho thiết kế responsive
  • Dễ dàng triển khai lên GitHub + Render

Dịch Vụ & Tính Năng

Những điểm nổi bật mà Lab3 mang lại

Giao Diện Hiện Đại

Thiết kế bằng Bootstrap, giúp trang web của bạn luôn “xịn sò” trên mọi thiết bị.

Backend Mỏng Nhẹ

Flask làm nhiệm vụ serve HTML tĩnh nên vô cùng nhẹ nhàng, dễ maintain.

Tốc Độ Tối Ưu

Chỉ load đúng CSS/JS cần thiết, hình ảnh được tối ưu, không có code thừa.
